## 三个核心玩法，不是噱头

### 户外日记

戴上 AirPods（或开启摄像头），向左右侧屈脖子，就能控制屏幕里的角色骑行、冲浪、划船……说起来有点像「用脖子打游戏」，但这个动作本身就是标准的颈部侧屈拉伸，活动一次等于做了一组针对性动作。

每次玩几分钟，沿途收集能量，解锁不同地图场景。清新唯美的画风加上舒缓配乐，整个体验意外地治愈。

### 坐姿守护

这个功能更日常一点。戴上 AirPods 或开启摄像头，它会实时检测你有没有低头。一旦发现你又开始「葛优躺」或者脑袋往前探，就会发出鸟鸣声提醒你——是那种百谷鸟的清脆叫声，不是那种让人心跳加速的警报。

保持良好坐姿一段时间，还会给你「果实」奖励。搞成了养成游戏的感觉，脑子会不自觉地想把果实集齐。

### 飞行之旅（iOS 专属）

这个模式需要配合 Apple Watch 使用，摆动手臂来控制角色飞行，解锁全球不同城市。手臂的摆动带动了整个上肢的活动，对缓解肩颈僵硬也有帮助。

值得一提的是，NeckGo 的科学顾问是中国康复医学会物理治疗专业委员会主任委员祁奇——主任物理治疗师、副教授级别。所以这不是拍脑袋想出来的「健康游戏」，背后有专业支撑。
